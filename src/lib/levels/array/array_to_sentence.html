Problem Description

Given an array of strings s, return a string with all of the strings in s in reverse order, separated by a space.

Examples

Input: s = ["world","Hello"]

Output: "Hello world"

Input: s = ["a","b","c"]

Output: "c b a"

Constraints

  • 1 <= s.length <= 100
  • 1 <= s[i].length <= 100
  • s[i] consists of printable ASCII characters.
