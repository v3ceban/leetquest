Array Basics

An array is a list of items stored at contiguous memory locations. It is a data structure that can hold multiple values of the same type.

Array elements are accessed using their index, which starts from 0. For example, in an array arr = [1, 2, 3], arr[0] will give you the first element (1), arr[1] will give you the second element (2), and so on.

Basic Syntax

Creating an Array

  • Empty Array: arr = []
  • Array with Elements: arr = [1, 2, 3]

Adding Elements

  • Append Element: arr.append(4)
  • Insert Element at Index: arr.insert(1, 5)
  • Extend Array: arr.extend([6, 7])

Accessing Elements

  • Accessing First Element: arr[0]
  • Accessing Second Element: arr[1]
  • Accessing Last Element: arr[-1]

Modifying Elements

  • Setting Element: arr[0] = 10
  • Direct Modification:
    • arr[1] += 5
    • arr[2] *= 2
    • arr[3] -= 1
    • arr[4] /= 2

Removing Elements

  • Remove Element by Value: arr.remove(2)
  • Remove Element by Index: arr.pop(1)
  • Clear Array: arr.clear()

Get Length of Array

  • len(arr)

Sort Array

  • arr.sort()
  • arr = sorted(arr)
