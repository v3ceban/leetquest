Receiving Input

Input for problems on LeetQuest use standard input and output. This means that you will need to read from the standard input stream and write to the standard output stream.

In Python, you can use the input() function to read one line of input from the standard input stream. The input is usually provided in a specific format, so you need to parse it accordingly.


Note: In the few problems we use standard input, we have the lines of code that read in all the inputs for you, but we figured it is still useful for you to know the important basics.

Input is a String

If the line of input contains a single string, you can read it like this:

input_string = input()

Input is an Integer

If the line of input contains a single integer, you read it in like a string and then have to type cast it to an integer:

input_integer = int(input())

Input is a List of Strings

If the line of input contains a list of strings, you can read it like this:

input_list = input().split()

This will split the input string by spaces and create a list of strings.


Input is a List of Integers

If the line of input contains a list of integers, you can read it like this:

input_list = list(map(int, input().split()))

This will split the input string by spaces, convert each element to an integer, and create a list of integers.

Writing Output

To write output to the standard output stream, you can use the print() function. For example:

print("Hello, World!")
