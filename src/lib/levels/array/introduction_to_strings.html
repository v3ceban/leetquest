String Basics

Strings are a sequence of characters enclosed in single or double quotes. They can be manipulated using various methods and functions.

Strings are immutable, meaning that once they are created, their contents cannot be changed. However, you can create new strings based on existing ones.


Basic Syntax

Creating a String

  • Single Quotes: string1 = 'Hello'
  • Double Quotes: string2 = "World"
  • Triple Quotes (for multi-line strings): string3 = '''Hello World'''

Accessing Characters

  • First Character: string1[0]
  • Second Character: string1[1]
  • Last Character: string1[-1]
  • Substring: string1[1:4]

String Length

  • len(string1)

String Concatenation

  • Using + operator: string1 + string2
  • Using join method: ''.join([string1, string2])
