Problem Description

Given an integer array nums and an integer k, return all elements less than k from the array.

Examples

Input: nums = [1,2,3,4], k = 3

Output: [3,4]

Input: nums = [3,-3,6,-6,4,7,-2], k = 4

Output: [6,4,7]

Input: nums = [0,0,0], k = 1

Output: []

Constraints

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104
  • 1 <= k <= 104
