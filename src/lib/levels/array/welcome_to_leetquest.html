What is LeetQuest?

LeetQuest is a platform that teaches you data structures and algorithms concepts and allows you to practice coding problems in a guided roadmap

The topics are split up into worlds, where each world teaches you the concept one subtopic at a time.

Level Nodes

Each level in any world is represented by a node:

  • When you complete a node, the connected nodes will unlock.
  • Learning nodes are denoted by blue nodes and will teach a subtopic of the world's concept.
  • Problem nodes are denoted by green, yellow, and red nodes, where the difficulty of the problem is easy, medium, and hard respectively.
    • Problems required to complete the world are denoted by numerical values.
    • Problems not required to complete the world (bonus problems) are denoted by uppercase letters. Bonus problems are extra problems that can be used as extra learning or a bonus challenge if done early.

Manual

The top right book icon is the manual, which is a popup that will contain all the content of unlocked nodes.

This can be used to refer to any of the content of previous nodes more conveniently.
