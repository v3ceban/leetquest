Problem Description

Given an array of distinct integers candidates and a target integer target, return all unique combinations of candidates where the chosen numbers sum to target. The numbers in candidates can be used an unlimited number of times in each combination.

Examples

Input: candidates = [2,3,4], target = 6

Output: [[2,2,2],[3,3],[2,4]]

Input: candidates = [2,3,5], target = 9

Output: [[2,2,2,3],[2,2,5],[3,3,3]]

Input: candidates = [3], target = 3

Output: [[3]]

Input: candidates = [4], target = 7

Output: []

Constraints

  • 1 <= candidates.length <= 30
  • 1 <= candidates[i] <= 200
  • candidates contains distinct integers.
  • 1 <= target <= 500
