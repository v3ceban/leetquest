Problem Description

Given an array of distinct integers candidates and a target integer target, return all unique combinations of candidates where the chosen numbers sum to target. The numbers in candidates can only be used once in each combination.

Examples

Input: candidates = [2,3,4], target = 6

Output: [[2,4]]

Input: candidates = [2,3,5], target = 9

Output: []

Input: candidates = [1,2,3,4,5], target = 5

Output: [[1,4],[2,3],[5]]]

Input: candidates = [4], target = 7

Output: []

Constraints

  • 1 <= candidates.length <= 30
  • 1 <= candidates[i] <= 200
  • candidates contains distinct integers.
  • 1 <= target <= 500
