Problem Description

Given two integers k and n, return all possible combinations of k numbers chosen from the range 1 to 9 that sum to n such that each number is used at most once.

Examples

Input: k = 3, n = 8

Output: [[1,2,5],[1,3,4]]

Input: k = 3, n = 9

Output: [[1,2,6],[1,3,5],[2,3,4]]

Input: k = 4, n = 9

Output: []

Constraints

  • 2 <= k <= 9
  • 1 <= n <= 60
