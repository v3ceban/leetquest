Problem Description

Given an array of distinct integers nums and a target integer target, return the number of possible combinations that add up to target.
Integers in nums can be used an unlimited number of times in one combination.
Different permutations of the same numbers count as different combinations.

Examples

Input: nums = [1,2,4], target = 4

Output: 6

Input: nums = [6], target = 3

Output: 0

Input: nums = [3], target = 3

Output: 1

Constraints

  • 1 <= nums.length <= 200
  • 1 <= nums[i] <= 1000
  • 1 <= target <= 1000
  • All elements of nums are distinct.
