Problem Description

Given an integer array nums that may contain duplicates, return all the possible permutations. You can return the answer in any order.

Examples

Input: nums = [1,1,2]

Output: [[1,1,2],[1,2,1],[2,1,1]]

Input: nums = [3,3,3]

Output: [[3,3,3]]

Input: nums = [0]

Output: [[0]]

Constraints

  • 1 <= nums.length <= 8
  • -106 <= nums[i] <= 106
