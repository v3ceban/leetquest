Problem Description

Given an integer n, return ALL of the structurally unique Binary Search Trees (BSTs) that have exactly n nodes of unique values from 1 to n.

Examples

Input: n = 1

Output: [[1]]

Input: n = 2

Output: [[1,null,2],[2,1]]

Input: n = 3

Output: [[1,null,2,null,3],[1,null,3,2],[2,1,3],[3,1,null,null,2],[3,2,null,1]]

Constraints

  • 1 <= n <= 8
