Problem Description

Given an array of integers nums sorted in non-decreasing order and an integer target, return the index of target in nums if it is present, or -1 if it is not present.
You must write an algorithm with O(log n) runtime complexity.

Examples

Input: nums = [1,2,4,6,7>,9], target = 7

Output: 4

Input: nums = [-1,0,1,3,5,8], target = 2

Output: -1

Constraints

  • 1 <= nums.length <= 105
  • -104 <= nums[i], target <= 104
  • nums is sorted in non-decreasing order.
