Problem Description

Given a sorted array of unique integers nums that has been rotated, return the minimum element of the array.
You must write an algorithm with O(log n) runtime complexity.

Examples

Input: nums = [6,8,0,2,4]

Output: 0

Input: nums = [1,2,3,4,5]

Output: 1

Constraints

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104
  • nums is sorted and rotated.
