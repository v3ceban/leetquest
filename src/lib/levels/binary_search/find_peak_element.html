Problem Description

Given a 0-indexed integer array nums, find a peak element, and return its index. A peak element is an element that is strictly greater than its neighbors. An element on the edge of the array only has to be adjacent to its only neighbor. If the array contains multiple peaks, return the index to any of the peaks.

Examples

Input: nums = [1,2,1]

Output: 1

Input: nums = [1,2,3,2,3,2,1,2,2,1]

Output: 2

Input: nums = [1,2,3,4,5,6,7]

Output: 6

Constraints

  • 1 <= nums.length <= 1000
  • -105 <= nums[i] <= 105
  • nums[i] != nums[i + 1] for all valid i
