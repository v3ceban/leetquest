Problem Description

You are working on a system with n versions of a software. Each version is represented by an integer from 1 to n. You are given a function isBadVersion(version) that returns true if the version is bad, and false otherwise. Your task is to find the first bad version.
You must write an algorithm that minimizes the number of calls to the API.

Examples

Input: n = 7, bad = 5

Execution:

> 7

                       4 <

> false

                       6 <

> true

                       5 <

> true

Output: 5

Constraints

  • 1 <= n <= 231 - 1
  • 1 <= bad <= n
