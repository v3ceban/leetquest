Problem Description

Given an array of integers piles, where piles[i] is the number of bananas in the i-th pile, and an integer h, representing the number of hours, return the minimum rate of bananas Koko should eat per hour (k) such that Koko can eat all the bananas within h hours.

Examples

Input: piles = [2,3,4,5,6], h = 8

Output: 3

Input: piles = [50,27,41,33,9], h = 5

Output: 50

Input: piles = [50,27,41,33,9], h = 6

Output: 41

Constraints

  • 1 <= piles.length <= 104
  • piles.length <= 109
  • 1 <= h <= 109
