Problem Description

Given an integer matrix matrix, with rows sorted in ascending order and the first element of each row being larger than the last one in the previous row, and an integer target, return true if target is in the matrix and false otherwise.
You must write an algorithm with O(log m * n) runtime complexity.

Examples

Input: matrix = [[1,3,5],[7,9,11],[12,13,14]], target = 9

Output: true

Input: matrix = [[1,3,5],[7,9,11],[12,13,14]], target = 8

Output: false

Constraints

  • m == matrix.length
  • n == matrix[i].length
  • 1 <= m, n <= 100
  • -104 <= matrix[i][j], target <= 104
  • All the integers in each row are distinct.
