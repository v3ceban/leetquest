What is Binary Search?

Binary search is an efficient algorithm used for finding a target value by repeatedly cutting the search space in half. It is applicable to sorted arrays or lists, where the elements are arranged in a specific order.

How Binary Search Works on a Structure:

  • Start with two pointers: left and right, representing the start and end of the search space.
  • Calculate the middle index: mid = (left + right) // 2.
  • If the target value is equal to the middle element, return the index.
  • If the target value is less than the middle element, adjust the right pointer to mid - 1.
  • If the target value is greater than the middle element, adjust the left pointer to mid + 1.
  • Moving the left and right pointers narrows down the search space by half each time.
  • Repeat until the target value is found or the search space is empty.
