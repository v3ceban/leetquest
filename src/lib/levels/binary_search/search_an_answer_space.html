Binary Search Outside of Structures

Binary search can also be used to search for a value within a certain range or to find the maximum or minimum value that satisfies a certain condition. This is often referred to as "searching an answer space." In this context, binary search is applied to a continuous range of values rather than a discrete set of elements.

How Binary Search Works on an Answer Space:

  • Start with two values: low and high, representing the low end and high end of the search space.
  • Calculate the middle value: mid = (low + high) // 2.
  • Check if the middle value is the desired answer.
  • If it is, return it.
  • If it is smaller than the desired answer, adjust the low pointer to mid + 1.
  • If it is larger than the desired answer, adjust the high pointer to mid - 1.
  • Repeat until the low pointer exceeds the high pointer or the desired answer is found.
