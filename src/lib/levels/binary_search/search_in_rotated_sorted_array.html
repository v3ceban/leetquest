Problem Description

Given a sorted array of unique integers nums that has been rotated and an integer target, return the index of target in nums if it is present, or -1 if it is not present.
You must write an algorithm with O(log n) runtime complexity.

Examples

Input: nums = [6,8,0,2,4], target = 0

Output: 2

Input: nums = [6,8,0,2,4], target = 3

Output: -1

Constraints

  • 1 <= nums.length <= 105
  • -104 <= nums[i], target <= 104
  • All integers in nums are unique.
  • nums is sorted and rotated.
