Problem Description

Given a sorted array of integers nums that has been rotated and an integer target, return true if target is in nums, or false if it is not present.
You must write an algorithm that reduces runtime complexity as much as possible.

Examples

Input: nums = [4,6,8,0,0,2,4], target = 0

Output: true

Input: nums = [4,6,8,0,0,2,4], target = 3

Output: false

Constraints

  • 1 <= nums.length <= 105
  • -104 <= nums[i], target <= 104
  • nums is sorted and rotated.
