Wait, I Don't Know What I am Looking For

In some binary search problems, the answer you are looking for is not explicitly given. Instead, you may need to find the maximum or minimum value that satisfies a certain condition.

How Binary Search Works on an Answer Space (Without Knowing the Answer):

  • Start with two values: low and high, representing the low end and high end of the search space.
  • Calculate the middle value: mid = (low + high) // 2.
  • If you are looking for a maximum value:
    • Check if the middle value satisfies the condition.
    • If it does, adjust the low pointer to mid to search for a larger value.
    • If it doesn't, adjust the high pointer to mid - 1 to cut off invalid values.
  • If you are looking for a minimum value:
    • Check if the middle value satisfies the condition.
    • If it does, adjust the high pointer to mid to search for a smaller value.
    • If it doesn't, adjust the low pointer to mid + 1 to cut off invalid values.
  • Repeat until the low pointer exceeds the high pointer or the desired answer is found.
