Problem Description

Given an integer array nums, return true if any value appears at least twice in the array, and return false if every element is distinct.

Example

Input: nums = [1,2,3,1]

Output: true

Input: nums = [1,2,3,4]

Output: false

Input: nums = [1,2,2]

Output: true

Constraints

  • 1 <= nums.length <= 105
  • -109 <= nums[i] <= 109
