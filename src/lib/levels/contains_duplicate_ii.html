Problem Description

Given an integer array nums and an integer k, return true if any value appears at least twice in the array within k indices from each other, and false otherwise.

Example

Input: nums = [1,2,3,1], k = 3

Output: false

Input: nums = [1,2,3,4], k = 10

Output: false

Input: nums = [2,1,2], k = 1

Output: false

Constraints
