Problem Description

Given an integer n, determine how many strings of length n can be formed using the vowels a, e, i, o, u under the following rules:

  • 'a' can only be followed by e
  • 'e' can only be followed by a or i
  • 'i' can only be followed by a, e, o, or u
  • 'o' can only be followed by i or u
  • 'u' can only be followed by a
Since the answer may be very large, return it modulo 109 + 7.

Examples

Input: n = 1

Output: 5

Input: n = 2

Output: 10

Input: n = 5

Output: 68

Constraints

  • 1 <= n <= 2 * 104
