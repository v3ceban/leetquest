Problem Description

Given a string s containing only digits, return the number of ways to decode it into letters. If the entire string cannot be decoded in any valid way, return 0.

Examples

Input: s = "115"

Output: 3

Input: s = "26"

Output: 2

Input: s = "01"

Output: 0

Constraints

  • 1 <= s.length <= 100
  • s[i] is a digit.
