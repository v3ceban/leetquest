Problem Description

Given two strings text1 and text2, return the minimum number of operations required to convert text1 to text2. The operations consist of inserting a character, deleting a character, or replacing a character.

Examples

Input: text1 = "position", text2 = "rotation"

Output: 3

Input: text1 = "battle", text2 = "war"

Output: 5

Input: text1 = "eleven", text2 = "twelve"

Output: 4

Constraints

  • 1 <= text1.length, text2.length <= 1000
  • text1 and text2 consist of lowercase English letters.
