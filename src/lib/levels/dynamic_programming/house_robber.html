Problem Description

Given an integer array nums representing the amount of money of each house, return the maximum amount of money you can rob tonight without alerting the police. Police are alerted if two adjacent houses are robbed.

Examples

Input: nums = [1,2,3,4,5]

Output: 9

Input: nums = [2,1,3,4]

Output: 6

Input: nums = [1,2]

Output: 2

Constraints

  • 1 <= nums.length <= 100
  • 0 <= nums[i] <= 400
