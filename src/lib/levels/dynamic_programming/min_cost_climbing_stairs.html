Problem Description

Given an integer array cost representing a staircase where cost[i] represents the cost of stepping on the i-th step, where on each step you can either climb 1 or 2 steps, return the minimum cost to reach the top of the stairs.

Examples

Input: cost = [5,10,15]

Output: 10

Input: cost = [1,100,1,2,1,100,1,2,100,1]

Output: 7

Constraints

  • 2 <= cost.length <= 1000
  • 0 <= cost[i] <= 999
