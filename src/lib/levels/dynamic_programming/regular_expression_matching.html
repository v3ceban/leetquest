Problem Description

Given an input string s and a pattern p, implement regular expression matching with support for '.' and '*', where '.' matches any single character and '*' matches zero or more of the preceding element.

Examples

Input: s = "star", p = ".ta*b*r"

Output: true

Input: s = "mississippi", p = "mis*is*p*."

Output: false

Input: s = "icepuzzles", p = ".*"

Output: true

Constraints

  • 0 <= s.length <= 20
  • 0 <= p.length <= 30
  • s can be empty and contains only lowercase letters.
  • p can be empty and contains only lowercase letters, '.', and '*'.
  • It is guaranteed that the input string and the pattern are both valid.
