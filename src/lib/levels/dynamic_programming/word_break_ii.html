Problem Description

Given a string s and a list of strings wordDict, return ALL of the ways s can be segmented into a space-separated sequence of one or more dictionary words.

Examples

Input: s = "leetquest", wordDict = ["leet","quest"]

Output: ["leet quest"]

Input: s = "penpineappleapplepen", wordDict = ["pine","apple","pen","pineapple"]

Output: ["pen pine apple apple pen","pen pineapple apple pen"]

Input: s = "combination", wordDict = ["comb","bin","nation"]

Output: []

Constraints

  • 1 <= s.length <= 300
  • 1 <= wordDict.length <= 1000
  • 1 <= wordDict[i].length <= 20
  • s and wordDict[i] consist of only lowercase English letters.
  • All the strings of wordDict are unique.
