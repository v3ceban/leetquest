What is Breadth First Search?

Breadth First Search (BFS) is a graph traversal algorithm that explores all the neighbors of a node before moving on to the next level of nodes. It is often implemented using a queue data structure.

The level order tree traversal algorithm uses BFS to visit all the nodes in a tree or graph.

BFS Process

  • Step 1: Initialize visited set. This set will store the nodes that the algorithm has already covered.
  • Step 2: Start at the root node. This is the first node that the algorithm will visit.
  • Step 3: Mark the current node as visited. This ensures that the algorithm does not visit this node multiple times.
  • Step 4: Visit all the neighbors of the current node that are NOT in the visited set. This is done by adding each neighbor to a queue.
  • Step 5: Dequeue the next node from the queue and repeat steps 3-4 until the queue is empty.

BFS Boilerplate

Here is a simple BFS implementation in Python:

def bfs(node):

    visited = set()

    queue = [node]

    while queue:

        current = queue.pop(0)

        if current not in visited:

            visited.add(current)

            for neighbor in current.neighbors:

                if neighbor not in visited:

                    queue.append(neighbor)
