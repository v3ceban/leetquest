Problem Description

Given an integer numCourses and an array prerequisites of pairs [a, b], where a is a prerequisite of b, return true if you can finish all courses. Otherwise, return false.

Examples

Input: numCourses = 3, prerequisites = [[1,0],[2,1]]

Output: true

Input: numCourses = 2, prerequisites = [[1,0],[0,1]]

Output: false

Constraints

  • 1 <= numCourses <= 2000
  • 0 <= prerequisites.length <= 5000
  • prerequisites[i].length == 2
  • 0 <= a, b < numCourses
  • All the pairs [a, b] are distinct.
