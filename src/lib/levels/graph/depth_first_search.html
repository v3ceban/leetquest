What is Depth First Search?

Depth First Search (DFS) is a graph traversal algorithm that explores as far as possible along each branch before backtracking. It is often implemented using recursion or a stack data structure.

The basic tree traversal algorithm uses DFS to visit all the nodes in a tree or graph.

DFS Process

  • Step 1: Initialize visited set. This set will store the nodes that the algorithm has already covered.
  • Step 2: Start at the root node. This is the first node that the algorithm will visit.
  • Step 3: Mark the current node as visited. This ensures that the algorithm does not visit this node multiple times.
  • Step 4: Visit all the neighbors of the current node that are NOT in the visited set. This is done by recursively calling the DFS function on each neighbor.
  • Step 5: Repeat steps 3-4 until all nodes have been visited.

DFS Boilerplate

Here is a simple DFS implementation in Python:

def dfs(node, visited):

    if node is None:

        return

    visited.add(node)

    for neighbor in node.neighbors:

        if neighbor not in visited:

            dfs(neighbor, visited)
