What is a Graph?

A graph is a data structure consisting of nodes, where each node has the following:

  • The value stored in the node
  • Edges that connect to other nodes

Graph Terminology

Graph Representations

Edge List

An edge list is a representation of a graph where each edge is represented as a pair of nodes in a list. This representation is simple and easy to understand, but it can be inefficient for large graphs.

Adjacency List

An adjacency list is a representation of a graph where each node is represented as a list of its neighbors. This representation is more efficient than an edge list for large graphs, as it allows for faster traversal and searching.

Adjacency Matrix

An adjacency matrix is a representation of a graph where each node is represented as a row and column in a matrix. The value at the intersection of a row and column indicates whether there is an edge between the two nodes. This representation is less efficient than an adjacency list for large graphs, as it requires more memory and can be slower to traverse.
