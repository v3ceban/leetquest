Problem Description

Given an array rooms, where rooms[i] is a list of keys you can obtain if you visit room i, return true if you can visit all rooms, or false otherwise. You start in room 0 and to enter other rooms, you need to have the keys to those rooms.

Examples

Input: rooms = [[2],[3],[1],[0]]

Output: true

Input: rooms = [[1,3],[3,0,4],[5,2,4],[0,1,5],[4,3],[4,1,5]]

Output: false

Constraints

  • n == rooms.length
  • 2 <= n <= 1000
  • 0 <= rooms[i].length <= 1000
  • 1 <= sum(rooms[i].length) <= 3000
  • 0 <= rooms[i][j] < n
  • All the values of rooms[i] are unique.
