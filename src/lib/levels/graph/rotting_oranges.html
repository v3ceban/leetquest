Problem Description

Given an m x n grid, where each cell can have one of three values:

  • 0 - representing an empty cell
  • 1 - representing a fresh orange
  • 2 - representing a rotten orange
Each minute, any fresh orange that is adjacent (4-directionally) to a rotten orange becomes rotten. Return the minimum number of minutes until no cell has a fresh orange. If this is impossible, return -1.

Examples

Input: grid = [
[2,1,1],
[0,1,0],
[1,1,2]]

Output: 2

Constraints

  • m == grid.length
  • n == grid[i].length
  • 1 <= m, n <= 300
  • grid[i][j] is either 0, 1, or 2.
