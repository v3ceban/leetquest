Problem Description

Given an m x n matrix board containing letters 'X' and 'O', capture regions of 'O' that are surrounded by 'X'. To capture a surrounding region, every 'O' in that region should be flipped to 'X'.

Examples

Input: grid = [
['O','X','X','O'],
['X','O','O','X'],
['X','X','O','X'],
['X','O','X','X']]

Output: grid = [
['O','X','X','O'],
['X','X','X','X'],
['X','X','X','X'],
['X','O','X','X']]

Constraints

  • m == grid.length
  • n == grid[i].length
  • 1 <= m, n <= 200
  • grid[i][j] is either 'X' or 'O'.
