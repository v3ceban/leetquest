Problem Description

Given two words beginWord and endWord, and a list wordList, find the length of the shortest transformation sequence from beginWord to endWord created from words in wordList (beginWord doesn't have to be in wordList), where every adjacent pair differs by a single letter. If there is no such transformation sequence, return 0.

Examples

Input: beginWord = "pot", endWord = "dig", wordList = ["pop","pug","put","dog","lot","log","cog","dig","cup"]

Output: 5

Input: beginWord = "dig", endWord = "cut", wordList = ["pop","pug","put","dog","lot","log","cog","dig","cup"]

Output: 0

Constraints

  • 1 <= beginWord.length, endWord.length <= 10
  • 1 <= wordList.length <= 1000
  • beginWord.length == endWord.length
  • beginWord, endWord, and wordList[i] consist of lowercase English letters.
  • beginWord != endWord
  • All the strings of wordList are unique.
