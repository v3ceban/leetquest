Problem Description

Given an array of strings strs, group the anagrams together.

Examples

Input: strs = ["add","sit","dad","not","its","ton"]

Output: [["add","dad"],["sit","its"],["not","ton"]]

Input: strs = [""]

Output: [[""]]

Input: strs = ["a","b"]

Output: [["a"],["b"]]

Constraints

  • 1 <= strs.length <= 104
  • 0 <= strs[i].length <= 100
  • strs[i] consists of lowercase English letters.
  • All inputs are guaranteed to be valid.
