Problem Description

Given an array nums of n integers where nums[i] is in the range [1, n], return an array of all the integers in the range [1, n] that do not appear in nums.

Examples

Input: nums = [5,3,1]

Output: [2,4]

Input: nums = [1,1,2,2]

Output: [3,4,5]

Constraints

  • n == nums.length
  • 1 <= n <= 105
  • 1 <= nums[i] <= n
