Mapping Elements to Indices

In some problems, it is helpful to store important indices of elements of a list in a hash map.

For example, if you have a list of elements and you want to store the last index of each element in a hash map, you can do it like this:

Example Code

hash_map = {}


for i in range(len(arr)):


    hash_map[arr[i]] = i


print(hash_map)

Example

For example, if you have the list arr = [1, 2, 3, 1, 2], the hash map would look like this:

> {1: 3, 2: 4, 3: 2}
