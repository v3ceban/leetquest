Problem Description

Convert a non-negative integer num to its English words representation.

Examples

Input: num = 345

Output: "Three Hundred Forty Five"

Input: num = 50500

Output: "Fifty Thousand Five Hundred"

Input: num = 9191919

Output: "Nine Million One Hundred Ninety One Thousand Nine Hundred Nineteen"

Constraints

  • 0 <= num <= 231 - 1
