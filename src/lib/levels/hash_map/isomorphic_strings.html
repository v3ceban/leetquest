Problem Description

Given two strings s and t, determine if they are isomorphic. Two strings s and t are isomorphic if the characters in s can be replaced to get t. No two characters may map to the same character, but a character may map to itself.

Examples

Input: s = "hello", t = "trees"

Output: true

Input: s = "add", t = "toe"

Output: false

Input: s = "toe", t = "add"

Output: false

Constraints

  • 1 <= s.length <= 5 * 104
  • t.length == s.length
  • s and t consist of any valid ASCII characters.
