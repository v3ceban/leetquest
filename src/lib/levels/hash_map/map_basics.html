What is a Hash Map?

A hash map is an unordered collection of items, each item consisting of a key and a value.

  • A hash map has no duplicate keys.

Basic Syntax

Initializing a Hash Map:

  • Empty Map: hash_map = {}
  • Non-Empty Set: hash_map = {"a": 1, "b": 2, "c": 3}

Setting a Key's Value:

  • hash_map["b"] = 4
    • > hash_map: {"a": 1, "b": 4, "c": 3}
  • The code will add the item if the key is not in the hash map yet.
  • The code will modify the key's value if the key is already in the hash map.
  • Note: The key must be a hashable type (e.g., strings, numbers, tuples).

Remove a Key:

  • hash_map.pop("a")
    • > hash_map: {"b": 4, "c": 3}
  • Note: The key must be in the hash map.

Checking if Key is in Hash Map:

  • if "b" in hash_map:
    • > true
  • if "a" in hash_map:
    • > false

Get a List of Keys:

  • keys = list(hash_map.keys())
    • keys: ["b", "c"]
  • Note: The order of the keys is not guaranteed.

Get a List of Values:

  • values = list(hash_map.values())
    • values: [4, 3]
  • Note: The order of the values is not guaranteed.

Get a List of Key-Value Pairs:

  • items = list(hash_map.items())
    • items: [("b", 4), ("c", 3)]
  • Note: The order of the items is not guaranteed.
