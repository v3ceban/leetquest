For Convenience

For some problems, it is nice to create your own hash map in order to create a mapping that is useful in the context of the problem.

Reminder

Initializing a Non-Empty Hash Map:

  • hash_map = {"key1": value1, "key2": value2, ...}

Example:

  • hash_map = {"one": 1, "two": 2, "three": 3}
    • > hash_map: {"one": 1, "two": 2, "three": 3}
