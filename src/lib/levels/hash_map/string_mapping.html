Mapping Strings to Other Strings

Other common uses for hash maps are mapping a string to another string.

For example, you can use a hash map to implement a substitution cipher, where each letter in the alphabet is replaced with another letter.

In this case, the key would be the letter in the alphabet, and the value would be the letter it is replaced with.

Example

If you want to replace the letter 'a' with 'b', you would add the following entry to the hash map:

  • hash_map['a'] = 'b'

Then, when you want to encrypt a message, you can iterate through the message and replace each letter with its corresponding value in the hash map.

If you want to encrypt the message "hello", you would do the following:

encrypted_message = ""


for letter in message:


    encrypted_message += hash_map[letter]


print(encrypted_message)

This would print the encrypted message, where each letter has been replaced with its corresponding value in the hash map.
