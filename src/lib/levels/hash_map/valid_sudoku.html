Problem Description

Given a 9x9 Sudoku board, determine if it is valid. A valid Sudoku board must have the digits 1 through 9 in each row, column, and 3x3 box without repetition.

Examples

Input: board =

[["5","3",".",".","7",".",".",".","."], ["6",".",".","1","9","5",".",".","."], [".","9","8",".",".",".",".","6","."], ["8",".",".",".","6",".",".",".","3"], ["4",".",".","8",".","3",".",".","1"], ["7",".",".",".","2",".",".",".","6"], [".","6",".",".",".",".","2","8","."], [".",".",".","4","1","9",".",".","5"], [".",".",".",".","8",".",".","7","9"]]

Output: true

Input: board =

[["5","3",".",".","7",".",".",".","9"], ["6",".",".","1","9","5",".",".","."], [".","9","8",".",".",".",".","6","."], ["8",".",".",".","6",".",".",".","3"], ["4",".",".","8",".","3",".",".","1"], ["7",".",".",".","2",".",".",".","6"], [".","6",".",".",".",".","2","8","."], [".",".",".","4","1","9",".",".","5"], [".",".",".",".","8",".",".","7","9"]]

Output: false

Constraints

  • board.length == 9
  • board[i].length == 9
  • board[i][j] is a digit or '.'
