What is a Heap?

A heap is a special tree-based data structure that satisfies the heap property. The heap property states that for a given node, the value of the node is less than or equal to the values of its children. This means that the smallest element is always at the root of the tree.

Heaps are commonly used to implement priority queues, where the highest priority element is always at the root of the tree. Heaps can be implemented using arrays or linked lists, but they are most commonly implemented using arrays.

Heap Operations

Heapify

Heapify is the process of rearranging the elements in a heap to maintain the heap property.

Heapify can be used on an array to transform it

  • heapq.heapify(arr) - Rearranges the elements in the array to maintain the heap property.

Push

Pushing is the process of adding a new element to the heap while maintaining the heap property.

Push can be used on a heap to add a new element to the heap.

  • heapq.heappush(arr, element) - Adds a new element to the heap while maintaining the heap property.

Pop

Popping is the process of removing the smallest element from the heap while maintaining the heap property.

Pop can be used on a heap to remove the smallest element from the heap.

  • heapq.heappop(arr) - Removes the smallest element from the heap while maintaining the heap property.
