Problem Description

Given an integer array nums and an integer k, return the kth largest element in the array without sorting the array.

Examples

Input: nums = [4,1,5,2,3], k = 2

Output: 4

Input: nums = [7,5,2,6,5,2,1,3,4], k = 4

Output: 5

Constraints

  • 1 <= nums.length <= 104
  • -104 <= nums[i] <= 104
  • 1 <= k <= n
