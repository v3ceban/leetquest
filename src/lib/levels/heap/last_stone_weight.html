Problem Description

Given an array of integers stones where stones[i] is the weight of the ith stone, repeat the following until at most one stone remains:

  • Find the two heaviest stones and smash them together.
  • The weight of the smashed stone is equal to the absolute difference of their weights.
  • If the smashed stone is not 0, put it back into the array.
  • Otherwise, remove both stones.
Return the weight of the last stone. If there are no stones left, return 0.

Examples

Input: stones = [16,8,8,4,2,1]

Output: 1

Input: stones = [1,3,2]

Output: 0

Input: stones = [3,7]

Output: 4

Constraints

  • 1 <= stones.length <= 30
  • 1 <= stones[i] <= 1000
