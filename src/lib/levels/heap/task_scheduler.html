Problem Description

Given an array of CPU tasks and an integer n, return the minimum number of CPU cycles needed to complete all tasks if all tasks take 1 CPU cycle to complete and n is the number of CPU cycles each task is on cooldown before it can be repeated.

Examples

Input: tasks = ["A","A","A","B","B"], n = 2

Output: 7

Input: tasks = ["A","A","A","B","B","C","C"], n = 2

Output: 8

Input: tasks = ["A","A","A"], n = 0

Output: 3

Constraints

  • 1 <= tasks.length <= 104
  • tasks[i] is upper-case English letter
  • 0 <= n <= 100
