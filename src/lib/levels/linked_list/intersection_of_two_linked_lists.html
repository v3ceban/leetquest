Problem Description

Given the heads of two linked lists headA and headB, return the node where the two lists intersect. If the two linked lists have no intersection at all, return null.

Examples

Input: headA = [4,1,8,4,5], headB = [5,0,1,8,4,5]

Output: 8

Input: headA = [1,2,3], headB = [4,5]

Output: null

Input: headA = [], headB = []

Output: null

Constraints

  • The number of nodes in the list is in the range [0, 103]
  • 1 <= Node.val <= 104
  • 0 <= ListNode.val <= 105
  • headA and headB do not have cycles.
