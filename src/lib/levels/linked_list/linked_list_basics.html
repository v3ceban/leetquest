What is a Linked List?

A linked list is a linear data structure consisting of objects called nodes

Each node has the following:

  • Data: The value stored in the node
  • Pointer: A reference to the next node in the list

The first node in the linked list is called the head, No nodes point to the head.

The last node in the linked list is called the tail. Since there are no nodes after it, the tail's pointer is set to null or None.

Traversing a Linked List

To traverse a linked list, start at the head and follow the pointers to each subsequent node until you reach the tail.

For example, given the following linked list:

head -> 1 -> 2 -> 3 -> 4 -> None

You can traverse it like this:

  • Start at the head (1)
  • Follow the pointer to the next node (2)
  • Follow the pointer to the next node (3)
  • Follow the pointer to the next node (4)
  • Follow the pointer to None, indicating the end of the list

In code, this can be done using a while loop:

current = head

while current is not None:

    print(current.data)

    current = current.next

Linked List Operations

Common operations on linked lists include:

  • Accessing: Retrieving the value of a node
  • Insertion: Adding a new node to the list
  • Deletion: Removing a node from the list

Accessing the Front of a Linked List

To access the front of a linked list, simply return the head node.

print(head.val)

Accessing the Back of a Linked List

To access the back of a linked list, traverse the list until you reach the tail node.

current = head

while current.next is not None:

    current = current.next

print(current.val)

Accessing Somewhere in the Middle of a Linked List

To access a node in the middle of a linked list, you can use a loop to traverse the list until you reach the desired index or value

current = head

index = 0

while current is not None and index < target_index:

    current = current.next

    index += 1

if current is not None:

    print(current.val)

else:

    print("Index out of range")

Inserting a Node at the Front of a Linked List

To insert a new node at the front of a linked list, create a new node and set its pointer to the current head. Then update the head to point to the new node.

new_node = Node(value)

new_node.next = head

head = new_node

Inserting a Node at the Back of a Linked List

To insert a new node at the back of a linked list, traverse the list until you reach the tail node. Then create a new node and set the tail's pointer to point to the new node.

current = head

while current.next is not None:

    current = current.next

new_node = Node(value)

current.next = new_node

Inserting a Node Somewhere in the Middle of a Linked List

To insert a new node somewhere in the middle of a linked list, traverse the list until you reach the desired index or value. Then create a new node and set its pointer to the current node's pointer. Finally, update the current node's pointer to point to the new node.

current = head

index = 0

while current is not None and index < target_index - 1:

    current = current.next

    index += 1

if current is not None:

    new_node = Node(value)

    new_node.next = current.next

    current.next = new_node

else:

    print("Index out of range")

Deleting a Node at the Front of a Linked List

To delete a node at the front of a linked list, simply update the head to point to the next node.

head = head.next

Deleting a Node at the Back of a Linked List

To delete a node at the back of a linked list, traverse the list until you reach the second-to-last node. Then set its pointer to None.

current = head

while current.next.next is not None:

    current = current.next

current.next = None

Deleting a Node Somewhere in the Middle of a Linked List

To delete a node somewhere in the middle of a linked list, traverse the list until you reach the desired index or value. Then update the previous node's pointer to skip the current node.

current = head

index = 0

while current is not None and index < target_index - 1:

    current = current.next

    index += 1

if current is not None and current.next is not None:

    current.next = current.next.next

else:

    print("Index out of range")
