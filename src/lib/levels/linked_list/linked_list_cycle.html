Problem Description

Given the head of a linked list, return true if it has a cycle (i.e., if there is some node in the list that can be reached again by continuously following the next pointer), or false otherwise.
DEV NOTE: ADD IMAGES BECAUSE THERE IS NO WAY TO EASILY INDICATE A LINK WITH TEXT

Examples

Input: head = [3,2,0,-4], pos = 1

Output: true

Input: head = [1,2], pos = 0

Output: true

Input: head = [1], pos = -1

Output: false

Constraints

  • The number of nodes in the list is in the range [0, 104]
  • -105 <= Node.val <= 105
  • pos is -1 or a valid index in the linked-list.
