Problem Description

Given a list lists of heads of sorted linked lists, merge them into one sorted linked list and return it.

Examples

Input: lists = [[1,4,5],[1,3,4],[2,6]]

Output: [1,1,2,3,4,4,5,6]

Input: lists = []

Output: []

Input: lists = [[]]

Output: []

Constraints

  • The number of nodes in all lists is in the range [0, 104]
  • The value of each node is in the range [-104, 104]
  • The input lists are sorted in ascending order.
