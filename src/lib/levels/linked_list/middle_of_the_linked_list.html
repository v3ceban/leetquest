Problem Description

Given the head of a linked list, return the middle node of the linked list. If there are two middle nodes, return the second one.

Examples

Input: head = [1,2,3]

Output: [2]

Input: head = [1,2,3,4]

Output: [3]

Input: head = []

Output: []

Constraints

  • The number of nodes in the list is in the range [0, 100]
  • 1 <= Node.val <= 100
