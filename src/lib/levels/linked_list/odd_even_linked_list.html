Problem Description

Given the head of a linked list, rearrange the nodes such that all odd-indexed nodes appear before all even-indexed nodes.
Note: In this problem, the index is 1-based.

Examples

Input: head = [1,2,3,4,5]

Output: [1,3,5,2,4]

Input: head = [2,1,3,5,6,4,7]

Output: [2,3,6,7,1,5,4]

Input: head = []

Output: []

Constraints

  • The number of nodes in the list is in the range [0, 104]
  • -5000 <= Node.val <= 5000
