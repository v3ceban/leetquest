Problem Description

Given the head of a linked list, return true if it is a palindrome or false otherwise.

Examples

Input: head = [1,2,1]

Output: true

Input: head = [1,2,2,1]

Output: true

Input: head = [1,2]

Output: false

Input: head = []

Output: true

Constraints

  • The number of nodes in the list is in the range [0, 105]
  • 0 <= Node.val <= 9
