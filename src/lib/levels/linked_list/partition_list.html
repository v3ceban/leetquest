Problem Description

Given the head of a linked list and an integer x, partition the list into two parts such that all nodes less than x come before nodes greater than or equal to x. Also preserve the original relative order of the nodes in each of the two partitions.

Examples

Input: head = [1,3,4,2,3,1], x = 3

Output: [1,2,1,3,4,3]

Input: head = [5,1], x = 5

Output: [1,5]

Input: head = [], x = 0

Output: []

Constraints

  • The number of nodes in the list is in the range [0, 200]
  • -100 <= Node.val <= 100
  • -100 <= x <= 100
