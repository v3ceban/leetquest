Problem Description

Given the head of a sorted linked list, remove all the duplicate nodes of the linked list, and return the list.

Examples

Input: head = [1,3,2,3,4,5]

Output: [1,3,2,4,5]

Input: head = [1,2,3,4]

Output: [1,2,3,4]

Input: head = [1,1]

Output: [1]

Constraints

  • The number of nodes in the list is in the range [0, 104]
  • 1 <= Node.val <= 50
