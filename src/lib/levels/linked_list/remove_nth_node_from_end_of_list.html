Problem Description

Given the head of a linked list, remove the nth node from the end of the list and return its head.

Examples

Input: head = [1,2,3,4,5], n = 4

Output: [1,3,4,5]

Input: head = [1], n = 1

Output: []

Input: head = [2,3,4], n = 2

Output: [2,4]

Constraints

  • The number of nodes in the list is in the range [1, 30]
  • 0 <= Node.val <= 100
  • 1 <= n <= 30
