Problem Description

Given the head of a linked list, rearrange the nodes in the linked list so that the index order is 1, n, 2, n - 1, 3, n - 2, ..., where n is the length of the linked list.

Examples

Input: head = [1,2,3,4,5]

Output: [1,5,2,4,3]

Input: head = [1,2,3,4,5,6]

Output: [1,6,2,5,3,4]

Input: head = []

Output: []

Constraints

  • The number of nodes in the list is in the range [0, 104]
  • -5000 <= Node.val <= 5000
