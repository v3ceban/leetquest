Problem Description

Given the head of a linked list, reverse the list, and return the reversed list.

Examples

Input: head = [1,2,3,4]

Output: [4,3,2,1]

Input: head = [1,3,2]

Output: [2,3,1]

Input: head = []

Output: []

Constraints

  • The number of nodes in the list is in the range [0, 5000]
  • -5000 <= Node.val <= 5000
