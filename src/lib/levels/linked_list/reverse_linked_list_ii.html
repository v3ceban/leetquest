Problem Description

Given the head of a linked list and two integers left and right, reverse the nodes of the linked list from position left to position right, and return the modified list.

Examples

Input: head = [1,2,3,4], left = 2, right = 4

Output: [1,4,3,2]

Input: head = [1,3,2], left = 2, right = 3

Output: [1,2,3]

Input: head = [1], left = 1, right = 1

Output: [1]

Constraints

  • The number of nodes in the list is in the range [0, 5000]
  • -5000 <= Node.val <= 5000
