Problem Description

Given the head of a linked list and an integer k, reverse the nodes of the linked list k nodes at a time, and return the modified list. If there are less than k nodes left, the remaining nodes should remain in the same relative order.

Examples

Input: head = [1,2,3,4], k = 2

Output: [2,1,4,3]

Input: head = [1,2,3,4], k = 3

Output: [3,2,1,4]

Input: head = [1,2,3,4,5,6,7,8], k = 3

Output: [3,2,1,6,5,4,7,8]

Constraints

  • The number of nodes in the list is in the range [0, 5000]
  • 1 <= Node.val <= 5000
  • 1 <= k <= 5000
