Problem Description

Given the head of a linked list and an integer k, rotate the list to the right by k places.

Examples

Input: head = [5,4,3,2,1], k = 1

Output: [1,5,4,3,2]

Input: head = [5,4,3,2,1], k = 3

Output: [3,2,1,5,4]

Input: head = [5,4,3,2,1], k = 5

Output: [5,4,3,2,1]

Input: head = [5,4,3,2,1], k = 7

Output: [2,1,5,4,3]

Constraints

  • The number of nodes in the list is in the range [0, 500]
  • -100 <= Node.val <= 100
  • 0 <= k <= 100
