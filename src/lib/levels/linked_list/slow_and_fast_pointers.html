What is a Fast Pointer?

A fast pointer is a pointer that moves at a faster speed than the slow pointer.

The following code shows how a fast pointer moves through a list:

current = head

while current is not None:

    current = current.next      # 1st Move

    if current is not None:     # Prevent errors 

        current = current.next  # 2nd Move

The if statement is there to ensure that current is not None when current.next is accessed
