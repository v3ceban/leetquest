Problem Description

Given the head of a linked list, swap every two adjacent nodes and return its head.

Examples

Input: head = [1,2,3,4]

Output: [2,1,4,3]

Input: head = [1,2,3]

Output: [2,1,3]

Input: head = []

Output: []

Constraints

  • The number of nodes in the list is in the range [0, 5000]
  • -5000 <= Node.val <= 5000
