Problem Description

Given an array nums of size n, return all elements that appear at least a third of the time.

Examples

Input: nums = [1,2,1,3]

Output: [1]

Input: nums = [1,2,3]

Output: [1,2,3]

Input: nums = [1,2,3,4]

Output: []

Constraints

  • n == nums.length
  • 1 <= n <= 5 * 104
  • -109 <= nums[i] <= 109
