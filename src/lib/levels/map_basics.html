What is a Hash Map?

A hash map is an unordered collection of items, each item consisting of a key and a value.

  • A hash map has no duplicate keys.

Basic Syntax

Initializing a Hash Map:

  • Empty Map: hash_map = {}
  • Non-Empty Set: hash_set = {"a": 1, "b": 2, "c": 3}

Setting a Key's Value:

  • hash_map["d"] = 4
  • The above code works even if the key is not in the hash map yet.

Remove a Key:

  • hash_map.pop("a")
