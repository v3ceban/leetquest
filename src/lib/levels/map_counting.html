Using a Hash Map to Count List Elements

A common use of hash maps is to count the number of times an element appears in a list.

When iterating through a list:

  • The first time you see an element, set its value to 1.
  • Every other time you see an element, increment its value by 1.

Example Code

hash_map = {}


for i in arr:


    if i in hash_map:


        hash_map[i] += 1


    else:


        hash_map[i] = 1
