Syntax

Set Union: All elements in set1 OR set2

  • set3 = set1 | set2

Set Intersection: All elements in set1 AND set2

  • set3 = set1 & set2

Set Difference: All elements in set1 that are NOT in set2

  • set3 = set1 - set2

Set Exclusive Union: All elements ONLY in set1 OR set2

  • set3 = set1 ^ set2
