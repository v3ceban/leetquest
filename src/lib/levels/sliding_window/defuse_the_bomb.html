Problem Description

Given a circular array code and an integer key k, return the decrypted array d to defuse the bomb, where d[i] is the sum of the next k elements in the circular array code. If k is negative, the sum is to be taken from the previous k elements in the circular array.

Examples

Input: code = [1,2,3,4,5], k = 3

Output: [9,12,10,8,6]

Input: code = [2,4,9,3], k = 0

Output: [0,0,0,0]

Input: code = [1,2,3,4], k = -1

Output: [4,1,2,3]

Constraints

  • 1 <= code.length <= 100
  • 1 <= code[i] <= 100
  • -(n - 1) <= k <= n - 1
