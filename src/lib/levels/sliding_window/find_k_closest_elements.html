Problem Description

Given a sorted integer array arr and two integers k and x, return the k closest integers to x in the array. The result should be sorted in ascending order. If two integers are equidistant from x, the smaller integer is preferred.

Examples

Input: arr = [1,2,3,4,5,6], k = 3, x = 3

Output: [2,3,4]

Input: arr = [1,2,3,4,5,6], k = 4, x = 3

Output: [1,2,3,4]

Input: arr = [1,2,3,4,5,6], k = 5, x = 3

Output: [1,2,3,4,5]

Input: arr = [1,2,3,4,5,6], k = 2, x = -3

Output: [1,2]

Constraints

  • 1 <= arr.length <= 104
  • arr.length == k
  • 1 <= k <= arr.length
  • -104 <= arr[i], x <= 104
  • arr is sorted in ascending order.
