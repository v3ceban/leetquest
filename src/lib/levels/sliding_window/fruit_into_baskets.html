Problem Description

Given the integer array fruits representing the type of fruits in a tree, return the maximum number of fruits you can pick up in two baskets where each basket can only pick up one type of fruit.

Examples

Input: fruits = [1,2,1,3]

Output: 3

Input: fruits = [0,0,1,2,1,2]

Output: 4

Constraints

  • 1 <= fruits.length <= 105
  • 0 <= fruits[i] < fruits.length
