Problem Description

Given a bookstore open for n minutes and the following:

  • An integer array customers of length n, where customers[i] is the number of customers that visit a bookstore on the ith minute.
  • An integer grumpy of length n, where grumpy[i] is 1 if the bookstore owner is grumpy and cannot satisfy customers on the ith minute, and 0 otherwise.
  • An integer minutes representing the number of minutes the bookstore owner can prevent themselves from being grumpy.

Return the maximum number of customers that can be satisfied while the bookstore is open.

Examples

Input: customers = [1,2,3,4,5], grumpy = [0,1,0,1,0], minutes = 1

Output: 13

Input: customers = [1,2,3,4,5,6,7], grumpy = [0,1,0,1,0,1,0], minutes = 3

Output: 26

Constraints

  • n == customers.length == grumpy.length
  • 1 <= n <= 2 * 104
  • 0 <= customers[i] <= 1000
  • 0 <= grumpy[i] <= 1
  • 1 <= minutes <= n
