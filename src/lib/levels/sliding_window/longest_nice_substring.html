Problem Description

Given a string s, return the first occurence of the longest substring where all letters in it appear both in uppercase and lowercase.

Examples

Input: s = "aAbcde"

Output: "aA"

Input: s = "aAbCde"

Output: "aA"

Input: s = "aAbCdDeEFfgh"

Output: "dDeEFf"

Constraints

  • 1 <= s.length <= 100
  • s consists of English letters
