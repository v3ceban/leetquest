Problem Description

Given a string s, return the length of the longest substring containing the same letter after replacing up to k characters with any letter.

Examples

Input: s = "ABABAA", k = 1

Output: 4

Input: s = "TTTDDS", k = 0

Output: 3

Input: s = "OKOK", k = 2

Output: 4

Constraints

  • 1 <= s.length <= 105
  • s consists of uppercase English letters.
  • 0 <= k <= s.length
