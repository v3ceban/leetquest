Problem Description

You are given an integer array nums of size n and an integer k, return the maximum average value of a contiguous subarray of size k rounded to 5 decimal places.

Examples

Input: nums = [1,2,3,4,5], k = 2

Output: 4.50000

Input: nums = [1,5,3,4,2], k = 3

Output: 4.00000

Input: nums = [3,2,0,4,0], k = 2

Output: 2.50000

Constraints

  • n == nums.length
  • 1 <= n <= 1000
  • 0 <= nums[i] <= 105
  • 1 <= k <= n
