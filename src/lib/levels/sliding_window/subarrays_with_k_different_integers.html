Problem Description

Given an integer array nums and an integer k, return the number of subarrays where there are exactly k different integers.

Examples

Input: nums = [3,1,2,1,2], k = 2

Output: 7

Input: nums = [3,1,2,1,3], k = 3

Output: 5

Input: nums = [1,2,1,2,1], k = 1

Output: 5

Constraints

  • 1 <= nums.length <= 20000
  • 1 <= nums[i] <= 100000
  • 1 <= k <= nums.length
