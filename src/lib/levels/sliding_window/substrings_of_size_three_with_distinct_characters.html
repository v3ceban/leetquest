Problem Description

Given a string s, return the number of substrings of length three that have no repeated characters.

Examples

Input: s = "abccac"

Output: 1

Input: s = "zxyxyxyzxyz"

Output: 5

Input: s = "ss"

Output: 0

Constraints

  • 1 <= s.length <= 100
  • s consists of lowercase English letters.
