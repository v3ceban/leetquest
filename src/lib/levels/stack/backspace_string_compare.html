Problem Description

Given two strings s and t, return true if they are equal when both are typed into empty text editors.
s and t contain the characters '#' which represents a backspace character. Note that after backspacing an empty text, the text will remain empty.

Examples

Input: s = "ab#c", t = "ad#c"

Output: true

Input: s = "ab###", t = "c##"

Output: true

Input: s = "a#b", t = "ab#"

Output: false

Constraints

  • 1 <= s.length, t.length <= 200
  • s and t only contain lowercase letters and '#' characters.
