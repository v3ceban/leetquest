Problem Description

Given a baseball game represented as a list of operations, where each operation can be:

  • "X": Record a new score x.
  • "+": Record a new score that is the sum of the previous two scores.
  • "D": Record a new score that is the double of the previous score.
  • "C": Remove previous score.
Return the sum of all the scores recorded after the operations.

Examples

Input: ["3", "1", "C", "D", "+"]

Output: 9

Input: ["5", "-2", "4", "C", "D", "9", "+"]

Output: 8

Constraints

  • 1 <= operations.length <= 1000
  • operations[i] is "+", "D", "C", or a string representing an integer.
  • The integer represented by operations[i] is between -1000 and 1000.
  • The operations are generated such that the sum of the elements in the stack will not exceed 231 - 1.
