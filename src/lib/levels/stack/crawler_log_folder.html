Problem Description

Given a list of string logs, return the minimum number of operations needed to go back to the main folder.

  • "X/" means go to folder X
  • "../" means go back one folder
  • "./" means stay in the same folder

Examples

Input: logs = ["d1/", "d2/", "../", "d21/", "./"]

Output: 2

Input: logs = ["d1/", "d2/", "./", "d3/", "../", "d31/"]

Output: 3

Constraints

  • 1 <= logs.length <= 100
  • 2 <= logs[i].length <= 100
  • logs[i] consist of lowercase English letters, digits, '.', and '/'.
