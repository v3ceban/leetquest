Problem Description

Given an encoded string, return its decoded string.
An encoded string will have the form k[s], where the s inside the square brackets is being repeated exactly k times. s can include other encoded strings.

Examples

Input: s = "2[ab]3[cd]ef"

Output: "ababcdcdcdef"

Input: s = "3[a2[c]]"

Output: "accaccacc"

Constraints

  • 1 <= s.length <= 30
  • s consists of lowercase English letters, digits, and square brackets [].
  • s is guaranteed to be a valid encoded string.
  • All the integers in s are in the range [1, 300].
