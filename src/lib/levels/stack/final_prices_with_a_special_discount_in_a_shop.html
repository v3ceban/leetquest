Problem Description

Given an integer array prices where prices[i] is the price of the i-th item in a shop, return an integer array answer where answer[i] is the final price you will pay for the ith item of the shop, considering a special discount where the price of the item is reduced by the first smaller price to the right of it, if any.

Examples

Input: prices = [8,4,6,2,3]

Output: [4,2,4,2,3]

Input: prices = [1,2,3,4,5]

Output: [1,2,3,4,5]

Input: prices = [5,4,3,2,1]

Output: [1,1,1,1,1]

Constraints

  • 1 <= prices.length <= 105
  • 1 <= prices[i] <= 104
