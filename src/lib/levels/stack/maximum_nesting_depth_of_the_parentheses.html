Problem Description

Given a string s of parentheses, return the maximum nesting depth of the parentheses in s.

Examples

Input: s = "(1)+((2))+(((3)))"

Output: 3

Input: s = "((1+(2*3)+((8)/4))+1)"

Output: 4

Constraints

  • 1 <= s.length <= 100
  • s[i] is either '(', ')', or a digit.
  • The parentheses are guaranteed to be balanced.
