Problem Description

Given an integer array nums, return the maximum width of a ramp in nums. A ramp in an integer array nums is a pair (i, j) for which i < j and nums[i] <= nums[j].
The width of such a ramp is j - i.

Examples

Input: nums = [6,0,8,2,1,5]

Output: 4

Input: nums = [9,8,1,0,1,9,4,0,4,1]

Output: 7

Constraints

  • 1 <= nums.length <= 5 * 105
  • 0 <= nums[i] <= 109
