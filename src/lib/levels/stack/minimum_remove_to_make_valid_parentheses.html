Problem Description

Given a parentheses string s, return the string after removing the minimum number of parentheses to make it valid.

Examples

Input: s = "lee(t(c)o)de)"

Output: "lee(t(c)o)de"

Input: s = "a)b(c)d"

Output: "ab(c)d"

Input: s = "))(("

Output: ""

Constraints

  • 1 <= s.length <= 105
  • s[i] is either '(' or ')' or lowercase English letter
