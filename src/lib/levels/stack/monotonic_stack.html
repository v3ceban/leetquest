What is a Monotonic Stack?

A monotonic stack is a stack that maintains its elements in a specific order, either strictly increasing or strictly decreasing. This property allows for efficient querying of the next greater or smaller element in a sequence.

When pushing onto a monotonic stack:

  • If the element being pushed is strictly larger/smaller than the top of the stack, it is pushed onto the stack.
  • If the element being pushed is smaller/larger than the top of the stack, pop elements from the stack until the above condition is met, then push the new element.

Popping from a monotonic stack doesn't have any special conditions since popping never changes the order of the elements in the stack.

Examples

stack = [1, 3, 4, 6]
  • stack.append(7)
  • Since 7 > 6, it is pushed onto the stack.
  • > stack: [1, 3, 4, 6, 7]
  • stack.append(2)
  • Since 2 < 7, pop elements from the stack until the top of the stack is less than 2, then push 2.
  • > stack: [1, 2]
  • stack.append(6)
  • Since 6 > 2, it is pushed onto the stack.
  • > stack: [1, 2, 6]
