Problem Description

Given two arrays nums1 and nums2, return an array answer such that answer[i] is the next greater element of nums1[i] in nums2 after its index in nums2. If it does not exist, return -1.

Examples

Input: nums1 = [4,1,2], nums2 = [1,3,4,2]

Output: [-1,3,-1]

Input: nums1 = [1,2,3,4], nums2 = [1,2,3,4]

Output: [2,3,4,-1]

Constraints

  • 1 <= nums1.length, nums2.length <= 1000
  • -104 <= nums1[i], nums2[i] <= 104
  • All integers in nums1 and nums2 are unique.
  • All the integers in nums2 are guaranteed to be present in nums1.
