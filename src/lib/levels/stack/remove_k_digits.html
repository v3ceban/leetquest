Problem Description

Given a string num representing a non-negative integer, and an integer k, return the smallest possible integer after removing k digits from num.

Examples

Input: num = "1432219", k = 3

Output: "1219"

Input: num = "10200", k = 1

Output: "200"

Constraints

  • 1 <= k <= num.length
  • num consists of digits and does not contain any leading zeros except for the zero itself.
  • num.length == n
  • 1 <= n <= 105
