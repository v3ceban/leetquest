Problem Description

Given a 0-indexed string word and a character ch, reverse the substring of word that starts at index 0 and ends at the index of the first occurrence of ch (inclusive). If the character ch does not exist in word, do nothing.

Examples

Input: word = "abcdef", ch = "d"

Output: "dcbaef"

Input: word = "xyxz", ch = "z"

Output: "zxyx"

Constraints

  • 1 <= word.length <= 100
  • word consists of lowercase English letters.
  • ch is a lowercase English letter.
