Problem Description

Given a balanced parentheses string s, return the score of the string which is based on the following rules:

  • "()" has a score of 1.
  • (A) has a score of 2 * score(A), where A is a balanced parentheses string.
  • AB has a score of score(A) + score(B), where A and B are balanced parentheses strings.

Examples

Input: s = "(())"

Output: 2

Input: s = "(())()"

Output: 3

Input: s = "(()())"

Output: 4

Constraints

  • 0 <= s.length <= 100
  • s[i] is either '(' or ')'
  • s is a balanced parentheses string.
