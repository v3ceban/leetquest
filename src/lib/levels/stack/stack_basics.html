What is a Stack?

A stack is an ordered list of elements where adding and removing elements always occurs at the same end of the list. This end is called the "top" of the stack. The other end is called the "bottom" of the stack.

A stack has two main operations:

  • push: Add an element to the top of the stack.
  • pop: Remove the top element from the stack.

Other operations include:

  • peek/top: Get the top element without removing it.
  • is_empty: Check if the stack is empty.
  • size: Get the number of elements in the stack.

Note: In Python, stacks are often implemented using arrays, where the top of the stack is at the end of the array, but you can also use the collections.deque module for better performance. The following syntax will only include the array implementation.

Basic Syntax

Initializing a Stack:

  • Empty Stack: stack = []
  • Non-Empty Stack: stack = [1, 2]

Pushing to a Stack:

  • stack.append(3)
  • > stack: [1, 2, 3]

Popping from a Stack:

  • stack.pop()
  • > stack: [1, 2]

Getting the Top Element:

  • top = stack[-1]
  • > top: 2

Checking if the Stack is Empty:

  • is_empty = len(stack) == 0
  • or is_empty = not stack
  • > is_empty: False

Getting the Size of the Stack: (Same Syntax as Array)

  • size = len(stack)
  • > size: 2
