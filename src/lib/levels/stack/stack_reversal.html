Using Stacks to Reverse Structures

In very rare cases, it can be useful to reverse structures using a list due to the nature of how stacks work.

For example, if you have a list of elements and you want to reverse the order of the elements, you can push each element onto a stack and then pop them off.

  • Due to the stack's inherent nature of first-in-last-out (FILO), the last element pushed onto the stack will be the first one popped off.
  • This means that after popping all the elements off the stack, the order of the elements will be reversed.

For example, if you have a stack stack = [1, 2, 3, 4], popping them off onto an array will give you:

> arr = [4, 3, 2, 1]
