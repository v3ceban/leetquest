With the Size of the Stack

In some stack problems, knowing the depth of nested parentheses or other structures that are best represented as a stack is important.

In these problems, it is a good idea to get the depth by getting the size of the stack using the len() function.

Example

If you wanted to count the number of letters in a string s at depth 1, you could do it like this:

count = 0


stack = []


for i in range(len(s)):


    if s[i] == "(":


        stack.append(s[i])


    elif s[i] == ")":


        stack.pop()


    else:


        if len(stack) == 1:


            count += 1


print("count = ", count)

If s = "a(b(c)d(e(f)g)h(i)j)k", the output would be:

> count = 4
