Problem Description

“Given a string s containing just the characters '(', '}', '{', '}', '[', and ']', determine if the input string is valid.”

Examples

Input: s = "()[]{}"

Output: true

Input: s = "({[]}}"

Output: true

Input: s = "(]"

Output: false

Input: s = "(()"

Output: false

Constraints

  • 1 <= s.length <= 104
  • s consists of only '()[]{}'
