Problem Description

Given an integer array nums and an integer k, return the k most frequent elements.

Examples

Input: nums = [1,1,1,2,2,3], k = 2

Output: [1,2]

Input: nums = [1], k = 1

Output: [1]

Constraints

  • 1 <= nums.length <= 105
  • k is in the range [1, the # of unique elements in nums]
  • -104 <= nums[i] <= 104
  • It is guaranteed that the answer is unique.
