Problem Description

Given the root of a binary tree, return true if it is a balanced binary tree, and false otherwise. A binary tree is balanced if the depth of the two subtrees of every node never differs by more than one.

Examples

Input: root = [1,2,3,4,5]

Output: true

Input: root = [1,2,null,3]

Output: false

Input: root = []

Output: true

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • -100 <= Node.val <= 100
