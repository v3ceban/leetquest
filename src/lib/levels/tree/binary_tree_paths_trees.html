Problem Description

Given the root of a binary tree, return ALL root-to-leaf paths.

Examples

Input: root = [1,2,3,4,5,6,7]

Output: [[1,2,4],[1,2,5],[1,3,6],[1,3,7]]

Input: root = [1,2,3]

Output: [[1,2],[1,3]]

Input: root = [1]

Output: [[1]]

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • 0 <= Node.val <= 100
