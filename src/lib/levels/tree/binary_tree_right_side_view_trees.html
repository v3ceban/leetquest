Problem Description

Given the root of a binary tree, return the nodes you would see if you were standing on the right side of the tree from top to bottom.

Examples

Input: root = [1,2,3,4,5,6,7]

Output: [1,3,7]

Input: root = [1,2,3,4,5]

Output: [1,3,5]

Input: root = []

Output: []

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • -100 <= Node.val <= 100
