Problem Description

Given the root of a complete binary tree, return the number of nodes in the tree.

Examples

Input: root = [1,2,3,4,5,6,7]

Output: 7

Input: root = [1,2,3]

Output: 3

Input: root = [1]

Output: 1

Constraints

  • The number of nodes in the tree is in the range [0, 5 * 104]
  • 0 <= Node.val <= 105
