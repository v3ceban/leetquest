Problem Description

Given the root of a binary tree, return all the modes of the tree, the values that appear most frequently in the tree.

Examples

Input: root = [1,2,2]

Output: [2]

Input: root = [1,2,2,1]

Output: [1,2]

Input: root = [0]

Output: [0]

Constraints

  • The number of nodes in the tree is in the range [1, 104]
  • 0 <= Node.val <= 109
