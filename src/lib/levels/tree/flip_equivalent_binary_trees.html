Problem Description

Given the roots of two binary trees root1 and root2, return true if the trees are flip equivalent or false otherwise. Two binary trees are considered flip equivalent if the left and right branches of any subtree of one tree can be swapped any number of times, and the resulting tree is identical to the other tree.

Examples

Input: root1 = [1,2,3], root2 = [1,3,2]

Output: true

Input: root1 = [1,2], root2 = [1,2]

Output: true

Input: root1 = [1,2,3,4], root2 = [1,3,2,4]

Output: false

Constraints

  • The number of nodes in both trees is in the range [0, 100]
  • -104 <= Node.val <= 104
