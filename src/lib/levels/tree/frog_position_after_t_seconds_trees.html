Problem Description

Given the edges of a tree with nodes numbered from 1 to n, and two integers t and target, determine the probability of a frog starting on root node 1 ending up on node target after t seconds. The frog jumps to any of its adjacent nodes every second if possible, otherwise it jumps in place. Answers within 10-5 of the actual answer will be accepted.

Examples

Input: edges = [[1,2],[1,3],[2,4],[2,5],[3,6],[3,7]], t = 2, target = 4

Output: 0.16666666666666666

Input: edges = [[1,2],[1,3],[2,4],[2,5],[3,6],[3,7]], t = 1, target = 7

Output: 0.33333333333333333

Input: edges = [[1,2],[1,3],[2,4],[2,5],[3,6],[3,7]], t = 0, target = 1

Output: 1.0

Constraints

  • 1 <= edges.length <= 1000
  • edges[i].length == 2
  • 1 <= edges[i][0], edges[i][1] <= n
  • edges[i][0] != edges[i][1]
  • 1 <= n <= 1000
  • 0 <= t <= 1000
  • 1 <= target <= n
