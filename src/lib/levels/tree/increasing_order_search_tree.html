Problem Description

Given the root of a binary tree, rearrange the tree into a linked list where the left child of each node should be set to null and the right child of each node points to the next node in the in-order traversal of the tree.

Examples

Input: root = [4,2,6,1,3,5,7]

Output: [1,null,2,null,3,null,4,null,5,null,6,null,7]

Input: root = [2,1,3]

Output: [1,null,2,null,3]

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • -100 <= Node.val <= 100
