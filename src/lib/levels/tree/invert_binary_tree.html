Problem Description

Given the root of a binary tree, return the inverted tree.

Examples

Input: root = [1,2,3]

Output: [1,3,2]

Input: root = [1,2,3,4]

Output: [1,3,2,null,null,null,4]

Input: root = []

Output: []

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • -100 <= Node.val <= 100
