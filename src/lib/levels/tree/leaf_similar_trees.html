Problem Description

Given the roots of two binary trees root1 and root2, return true if both trees, when iterated through using inorder traversal, access the same leaf nodes in the same order.

Examples

Input: root1 = [1,2,3], root2 = [1,2,3]

Output: true

Input: root1 = [1,3,2], root2 = [1,2,2,3]

Output: true

Input: root1 = [1,2,null,3], root2 = [1,null,3,null,2]

Output: false

Constraints

  • The number of nodes in both trees is in the range [0, 2000]
  • -104 <= Node.val <= 104
