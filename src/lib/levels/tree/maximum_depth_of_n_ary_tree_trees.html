Problem Description

Given the root of an n-ary tree, return its maximum depth, the number of nodes along the longest path from the root node down to the farthest leaf node.

Examples

Input: root = [1,null,2,3,4,5,null,6,7,null,8,9]

Output: 3

Input: root = [1,null,2,3,4,null,null,5,null,6,null,7]

Output: 4

Constraints

  • The number of nodes in the tree is in the range [0, 104]
  • 0 <= Node.val <= 100
  • The depth of the n-ary tree is less than or equal to 1000
