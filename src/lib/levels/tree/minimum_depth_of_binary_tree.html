Problem Description

Given the root of a binary tree, return its minimum depth, the number of nodes along the longest path from the root node down to the closest leaf node.

Examples

Input: root = [1,2,3,4,5]

Output: 2

Input: root = [1,2,3]

Output: 2

Input: root = []

Output: 0

Constraints

  • The number of nodes in the tree is in the range [0, 105]
  • -100 <= Node.val <= 100
