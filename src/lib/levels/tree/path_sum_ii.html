Problem Description

Given the root of a binary tree and an integer targetSum, return ALL the root-to-leaf paths where the sum of the values along the path equals targetSum.

Examples

Input: root = [1,2,4,5,6,3,7], targetSum = 8

Output: [[1,2,5],[1,4,3]]

Input: root = [1,2,3], targetSum = 5

Output: []

Input: root = [1], targetSum = 1

Output: [[1]]

Constraints

  • The number of nodes in the tree is in the range [0, 5000]
  • -1000 <= Node.val <= 1000
  • -1000 <= targetSum <= 1000
