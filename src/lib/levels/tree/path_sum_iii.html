Problem Description

Given the root of a binary tree and an integer targetSum, return the number of downwards paths where the sum of the values along the path equals targetSum.

Examples

Input: root = [1,2,4,6,7,4,5], targetSum = 9

Output: 4

Input: root = [1,2,3], targetSum = 5

Output: 0

Input: root = [1], targetSum = 1

Output: 1

Constraints

  • The number of nodes in the tree is in the range [0, 1000]
  • -1000 <= Node.val <= 1000
  • -1000 <= targetSum <= 1000
