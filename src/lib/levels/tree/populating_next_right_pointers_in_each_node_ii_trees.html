Problem Description

Given the root of a binary tree, add next right pointers to each node in the tree that point to the next node on the same level. The right pointer for the rightmost node in each level should be set to null.

Examples

Input: root = [1,2,3,4,5]

Output: [1,#,2,3,#,4,5,#]

Input: root = [1,2,3,4,null,null,5,null,6]

Output: [1,#,2,3,#,4,5,#,6,#]

Input: root = []

Output: []

Constraints

  • The number of nodes in the tree is in the range [0, 6000]
  • All nodes have val in the range [-100, 100]
