Problem Description

Given the root of a perfect binary tree (all leaves are on the same level), add next right pointers to each node in the tree that point to the next node on the same level. The right pointer for the rightmost node in each level should be set to null.

Examples

Input: root = [1,2,3,4,5,6,7]

Output: [1,#,2,3,#,4,5,6,7,#]

Input: root = [1,2,3]

Output: [1,#,2,3,#]

Input: root = []

Output: []

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • All nodes have val in the range [-100, 100]
  • The tree is a perfect binary tree.
