Problem Description

Given the root of a binary search tree and two integers low and high, return the sum of values of all nodes with a value in the inclusive range [low, high].

Examples

Input: root = [4,2,6,1,3,5,7], low = 2, high = 6

Output: 20

Input: root = [1], low = 1, high = 1

Output: 1

Input: root = [1], low = 0, high = 0

Output: 0

Constraints

  • The number of nodes in the tree is in the range [1, 2 * 104]
  • 0 <= Node.val <= 105
  • The value of each node in the tree is unique.
  • root is guaranteed to be a valid binary search tree.
  • 0 <= low <= high <= 105
