Problem Description

Given the roots of two binary trees p and q, return true if they are the same tree, or false if they are not. Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.

Examples

Input: p = [1,null,2,3], q = [1,null,2,3]

Output: true

Input: p = [1,2,3], q = [1,2,null,3]

Output: false

Constraints

  • The number of nodes in both trees is in the range [0, 100]
  • -104 <= Node.val <= 104
