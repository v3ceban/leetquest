Problem Description

Given the root of a binary search tree and an integer val, return the node with the value val in the binary search tree, or null if the value does not exist in the tree.

Examples

Input: root = [4,2,7,1,3], val = 2

Output: [2,1,3]

Input: root = [4,2,7,1,3], val = 5

Output: []

Constraints

  • The number of nodes in the tree is in the range [0, 104]
  • 1 <= Node.val <= 107
  • root is a binary search tree.
  • 1 <= val <= 107
