Problem Description

Given the root of a binary tree where the value of each non-leaf node is equal to the minimum value of its children, return the second minimum value in the tree. If no such value exists, return -1.

Examples

Input: root = [2,2,4,2,3,4,5]

Output: 3

Input: root = [1,1,1]

Output: -1

Constraints

  • The number of nodes in the tree is in the range [1, 104]
  • 1 <= Node.val <= 105
  • The value of each non-leaf node is equal to the minimum value of its children.
