Problem Description

Given the root of a binary tree, return the sum of all of the left leaves.

Examples

Input: root = [1,2,3]

Output: 2

Input: root = [1]

Output: 0

Input: root = [1,2,3,4,5,6,7]

Output: 10

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • -1000 <= Node.val <= 1000
