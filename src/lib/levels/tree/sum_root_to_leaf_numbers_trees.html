Problem Description

Given the root of a binary tree, return the sum of all root-to-leaf numbers. A root-to-leaf number is formed by concatenating the values of a root-to-leaf path.

Examples

Input: root = [1,2,3]

Output: 25

Input: root = [1]

Output: 1

Input: root = [1,2,3,4,5,6,7]

Output: 522

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • 0 <= Node.val <= 9
