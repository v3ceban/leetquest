Problem Description

Given the root of a binary tree, return true if it is a mirror of itself (i.e., symmetric around its center) and false otherwise.

Examples

Input: root = [1,2,2,null,3,3]

Output: true

Input: root = [1,2,2,3,3]

Output: false

Constraints

  • The number of nodes in the tree is in the range [0, 100]
  • -104 <= Node.val <= 104
