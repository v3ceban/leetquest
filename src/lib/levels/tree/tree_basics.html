What is a Tree?

A tree is a data structure consisting of objects called nodes

Each node has the following:

  • The value stored in the node
  • Two pointers, left and right, to different nodes in the tree

The first node in the tree is called the root, No nodes point to the root.

Nodes that have pointers to other nodes have child nodes.

Nodes that are pointed to by other nodes have parent nodes.

Nodes that have no child nodes are called leaf nodes.

Traversing a Tree

If we try to traverse a tree like we did with a linked list, we will run into a problem. If we follow a branch in a tree to its leaf node, we will not be able to go back up the tree to follow another branch.

To traverse a tree, we will have to use recursion to backtrack to previous nodes we have visited.

For example, given the following tree:

1
2 3
4 5 6 7

We can traverse it like this:

  • Start at the root (1)
  • Follow the left pointer to the next node (2)
  • Follow the left pointer to the next node (4)
  • Since 4 has no children, backtrack to 2
  • Follow the right pointer to the next node (5)
  • Since 5 has no children, backtrack to 2
  • Since we have visited all children of 2, backtrack to 1
  • Follow the right pointer to the next node (3)
  • Follow the left pointer to the next node (6)
  • Since 6 has no children, backtrack to 3
  • Follow the right pointer to the next node (7)
  • Since 7 has no children, backtrack to 3
  • Since we have visited all children of 3, backtrack to 1
  • Since we have visited all children of 1, we are done

Brief Lesson in Recursion

Recursion is a programming technique where a function calls itself to solve a problem.

Recursion is often used to solve problems that can be broken down into smaller subproblems, in this instance, breaking down a tree into smaller subtrees.

To use recursion, we need to define a base case and a recursive case.

The base case is the condition that stops the recursion. In this case, the base case is when we reach null.

The recursive case is the condition that continues the recursion. In this case, the recursive case is when we follow the left and right pointers to the next nodes.

In code, this can be done using a function that calls itself:

def traverse_tree(node):

    # Base case: node is null

    if node is None:

        return

    
    # Recursive case: follow children

    traverse_tree(node.left)

    traverse_tree(node.right)

Different Tree Traversals

Preorder Traversal

  • 1. Track the current node
  • 2. Recurse on the left subtree
  • 3. Recurse on the right subtree

Inorder Traversal

  • 1. Recurse on the left subtree
  • 2. Track the current node
  • 3. Recurse on the right subtree

Postorder Traversal

  • 1. Recurse on the left subtree
  • 2. Recurse on the right subtree
  • 3. Track the current node
