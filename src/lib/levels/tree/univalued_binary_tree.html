Problem Description

Given the root of a binary tree, return true if all the nodes in the tree have the same value, and false otherwise.

Examples

Input: root = [1,1,null,1,1]

Output: true

Input: root = [1,1,2]

Output: false

Input: root = [2,1,1]

Output: false

Constraints

  • The number of nodes in the tree is in the range [1, 100]
  • 0 <= Node.val <= 100
