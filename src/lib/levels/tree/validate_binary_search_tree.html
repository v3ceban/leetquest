Problem Description

Given the root of a binary tree, return true if it is a valid binary search tree (BST), and false otherwise.

Examples

Input: root = [2,1,3]

Output: true

Input: root = [2,1,3,4]

Output: false

Input: root = [3,1,5,null,4]

Output: false

Constraints

  • The number of nodes in the tree is in the range [0, 104]
  • -231 <= Node.val <= 231 - 1
