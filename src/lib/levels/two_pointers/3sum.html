Problem Description

Given an array of integers nums, return a list of all unique triplets that sum to 0.

Examples

Input: nums = [-2,2,0,1,-1]

Output: [[-2,2,0],[0,1,-1]]

Input: nums = [1,1,1]

Output: []

Input: nums = [0,0,0,0,0,0]

Output: [[0,0,0]]

Constraints

  • 0 <= nums.length <= 3000
  • -105 <= nums[i] <= 105
