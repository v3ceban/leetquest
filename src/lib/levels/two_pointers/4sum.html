Problem Description

Given an array of integers nums and an integer target, return a list of all unique quadruplets that sum to target.

Examples

Input: nums = [-2,2,0,1,-1], target = 0

Output: [-2,2,1,-1]

Input: nums = [1,1,1], target = 3

Output: []

Input: nums = [0,0,0,0,0,0], target = 0

Output: [0,0,0,0]

Constraints

  • 0 <= nums.length <= 3000
  • -105 <= nums[i] <= 105
  • -109 <= target <= 109
