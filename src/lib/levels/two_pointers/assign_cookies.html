Problem Description

Given an integer array g representing each child's greed factor and integer array s representing the size of each cookie, return the maximum number of children who can be content with their cookies. A child is content if their greed factor is less than or equal to the size of the cookie they receive.

Examples

Input: g = [1,2,3], s = [1,1,3]

Output: 2

Input: g = [1,2,3,4], s = [1,2,3,4,5,6]

Output: 4

Constraints

  • 1 <= g.length, s.length <= 3 * 104
  • 1 <= g[i], s[i] <= 231 - 1
