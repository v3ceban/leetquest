Problem Description

Given two version strings version1 and version2, determine which version is the latest one.

  • If version1 > version2, return 1.
  • If version2 > version1, return -1.
  • If both are the same, return 0.

Examples

Input: version1 = "1.1", version2 = "0.1"

Output: 1

Input: version1 = "1.01", version2 = "1.001"

Output: 0

Input: version1 = "1.0", version2 = "1.0.0.1"

Output: -1

Constraints

  • 1 <= version1.length, version2.length <= 15
  • version1 and version2 only contain digits and '.'.
  • version1 and version2 are valid version numbers.
  • All the given version numbers are non-negative integers, and their leading zeros are ignored.
