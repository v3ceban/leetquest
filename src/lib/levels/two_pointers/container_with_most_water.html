Problem Description

Given an array if integers height representing the height of walls, where the width between each pair of adjacent walls is 1, find the largest container that two walls form together with the x-axis.

Examples

Input: height = [1,4,3,2,1,6,2]

Output: 16

Input: height = [2,2]

Output: 2

Constraints

  • 2 <= height.length <= 105
  • 0 <= height[i] <= 104
