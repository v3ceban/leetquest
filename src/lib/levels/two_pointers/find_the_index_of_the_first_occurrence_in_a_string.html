Problem Description

Given two strings needle and haystack, return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Examples

Input: haystack = "mississippi", needle = "is"

Output: 1

Input: haystack = "observation", needle = "observe"

Output: -1

Constraints

  • 1 <= haystack.length, needle.length <= 100
  • haystack and needle consist of only lowercase English letters.
  • Return -1 if needle is not part of haystack.
