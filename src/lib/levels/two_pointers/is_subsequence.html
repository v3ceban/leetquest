Problem Description

Given two strings s and t, return true if s is a subsequence of t, or false otherwise.

Examples

Input: s = "cab", t = "scarab"

Output: true

Input: s = "axe", t = "scarab"

Output: false

Input: s = "axe", t = "exactly"

Output: false

Constraints

  • 0 <= s.length <= 100
  • 0 <= t.length <= 104
  • s and t consist of lowercase English letters.
