Problem Description

Given two integer arrays nums1 and nums2 and their lengths m and n, merge nums2 into nums1 as one sorted array.

Examples

Input: nums1 = [1,4,6,0,0,0], m = 3, nums2 = [2,3,5], n = 3

Output: [1,2,3,4,5,6]

Input: nums1 = [1], m = 1, nums2 = [], n = 0

Output: [1]

Input: nums1 = [0], m = 0, nums2 = [1], n = 1>

Output: [1]

Constraints

  • nums1.length == m + n
  • nums2.length == n
  • 0 <= m, n <= 200
  • 1 <= m + n <= 200
  • -109 <= nums1[i], nums2[i] <= 109
