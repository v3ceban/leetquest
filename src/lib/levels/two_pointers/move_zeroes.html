Problem Description

Given an integer array nums, move all 0s to the end of it while maintaining the relative order of the non-zero elements.

Examples

Input: nums = [0,1,0,2,3]

Output: [1,2,3,0,0]

Input: nums = [0,0,4,0,0,0]

Output: [4,0,0,0,0,0]

Constraints

  • 1 <= nums.length <= 104
  • -231 <= nums[i] <= 231 - 1
