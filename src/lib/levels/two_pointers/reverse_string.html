Problem Description

Given a string represented as an array, reverse it.

Examples

Input: s = ["h", "e", "l", "l", "o"]

Output: ["o", "l", "l", "e", "h"]

Input: s = ["l", "e", "v", "e", "l"]

Output: ["l", "e", "v", "e", "l"]

Constraints

  • 1 <= s.length <= 105
  • s consists of printable ASCII characters.
