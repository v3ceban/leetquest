Problem Description

Given a string s and a character c, return an array of integers answer such that answer[i] is the distance from index i to the closest occurrence of character c in the string.

Examples

Input: s = "mississippi", c = "s"

Output: [2,1,0,0,1,0,0,1,2,3,4]

Input: s = "abcdefedcba", c = "a"

Output: [0,1,2,3,4,5,4,3,2,1,0]

Constraints

  • 1 <= s.length <= 104
  • s[i] and c are lowercase English letters.
  • It can be proven that answer is unique.
