Problem Description

Given an integer array nums, reorder the array such that all even integers appear before all odd integers. Return any array that satisfies this condition.

Examples

Input: nums = [1,2,3,4]

Output: [4,2,3,1]

Input: nums = [5,1,7,2]

Output: [2,1,7,5]

Constraints

  • 1 <= nums.length <= 5000
  • 0 <= nums[i] <= 5000
