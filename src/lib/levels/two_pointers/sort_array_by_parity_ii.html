Problem Description

Given an integer array nums where half the integers are even and the other half are odd, reorder the array such that all even integers appear on even indices and all odd integers appear on odd indices. Return any array that satisfies this condition.

Examples

Input: nums = [4,2,5,7]

Output: [4,5,2,7]

Input: nums = [1,2,3,4]

Output: [2,1,4,3]

Constraints

  • 1 <= nums.length <= 5000
  • 0 <= nums[i] <= 5000
