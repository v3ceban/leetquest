Problem Description

Given an array nums of 3 colors, represented as integers 0, 1, and 2, sort the array in-place so that objects of the same color are adjacent, with the colors in the order green (0), blue (1), and purple (2).

Examples

Input: nums = [2,0,2,1,1,0]

Output: [0,0,1,1,2,2]

Input: nums = [2,0,1,2,0,1]

Output: [0,0,1,1,2,2]

Constraints

  • 1 <= nums.length <= 300
  • nums[i] is either 0, 1, or 2.
