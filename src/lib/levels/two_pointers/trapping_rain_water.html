Problem Description

Given an integer array height representing an elevation map where the width of each bar is 1, return how much water it can trap after raining.

Examples

Input: height = [0,1,0,2,1,1,2,3,1,0,2]

Output: 6

Input: height = [1,2,3,4,5]

Output: 0

Input: height = [5,4,3,2,1,5,5]

Output: 10

Constraints

  • n == height.length
  • 0 <= n <= 2 * 104
  • 0 <= height[i] <= 105
