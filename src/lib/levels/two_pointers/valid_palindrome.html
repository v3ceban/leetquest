Problem Description

Given a string s, determine if it is a palindrome, considering only alphanumeric characters and ignoring cases.

Examples

Input: s = "Race car"

Output: true

Input: s = "solid amadilos"

Output: true

Input: s = "Symbols! make (THIS) PrObLeM |||unreasonably||| $annoying$"

Output: false

Constraints

  • 1 <= s.length <= 23
  • s consists only of printable ASCII characters.
