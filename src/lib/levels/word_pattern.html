Problem Description

Given a pattern and a string s, find if s follows the same pattern. s follows the pattern if the characters in pattern can be replaced with words to get s.

Examples

Input: pattern = "abca", s = "pen pineapple apple pen"

Output: true

Input: pattern = "abba", s = "pen pineapple apple pen"

Output: false

Input: pattern = "aaa", s = "one two three"

Output: false

Constraints

  • 1 <= pattern.length <= 300
  • pattern contains only lowercase English letters.
  • 1 <= s.length <= 3000
  • s contains only lowercase English letters and spaces ' '.
  • s does not contain any leading or trailing spaces.
  • All the words in s are separated by a single space.
